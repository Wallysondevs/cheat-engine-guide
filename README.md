# Guia Cheat Engine 🎮

  Guia completo do **Cheat Engine** em Português Brasileiro — 30 tópicos cobrindo do básico ao avançado.

  ## 📚 Conteúdo

  - **Fundamentos** — O que é o CE, instalação (Windows e Linux), interface, tipos de dados
  - **Varredura de Memória** — Primeira varredura, valor desconhecido, tipos de scan, refinamento
  - **Gerenciamento de Endereços** — Lista, freeze, modificar valores
  - **Ponteiros** — Conceitos, encontrar manualmente, Pointer Scanner, multinível
  - **Debugger e Disassembler** — Memory View, disassembler, breakpoints, registradores
  - **Code Injection** — Auto-Assemble, injeção de código, templates
  - **Lua Scripting** — Introdução, API do CE, scripts automáticos, hotkeys
  - **Tabelas e Trainers** — Cheat Tables, Speed Hack, criar trainers
  - **Anti-Cheat e Proteções** — Como funcionam, técnicas (fins educacionais)
  - **Tutorial Oficial** — Todos os 9 passos explicados em Português
